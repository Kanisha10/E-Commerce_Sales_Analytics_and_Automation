# E-Commerce_Sales_Analytics_and_Automation
## Project Overview

This project focuses on automating the process of cleaning, preprocessing, and analyzing e-commerce sales data using Python and creating interactive business reports using Power BI. The goal is to improve data quality, eliminate inconsistencies, and generate meaningful insights through automated reporting and visualization.

---

## Objectives

* Handle missing values in the dataset
* Remove duplicate records
* Correct inconsistent data formats
* Automate data preprocessing using Python
* Generate clean datasets for analysis
* Create interactive dashboards and reports in Power BI
* Provide business insights through visual summaries

---

## Dataset

The project uses an e-commerce sales dataset containing information such as:

* Order Number
* Product Line
* Customer Name
* Sales
* Quantity Ordered
* Order Date
* Country
* Territory
* Deal Size

---

## Technologies Used

### Python

* Pandas
* NumPy
* OpenPyXL

### Data Visualization

* Matplotlib
* Seaborn

### Business Intelligence

* Power BI

---

## Data Cleaning Process

### Missing Value Handling

* Filled missing address information
* Replaced null state, postal code, and territory values with default labels

### Duplicate Removal

* Identified and removed duplicate records

### Inconsistent Data Correction

* Standardized product line names
* Standardized country names
* Cleaned customer name formatting
* Converted date fields into proper datetime format

---

## Key Performance Indicators (KPIs)

The dashboard includes:

* Total Sales
* Total Orders
* Total Customers
* Sales by Product Line
* Sales by Country
* Sales by Territory
* Monthly Sales Trends
* Top Customers
* Deal Size Distribution

---

## Power BI Dashboard Pages

### Executive Summary

* Total Sales
* Total Orders
* Total Customers
* Monthly Revenue Trend

### Product Analysis

* Product Line Performance
* Revenue by Product Category

### Regional Analysis

* Country-wise Sales
* Territory-wise Sales

### Customer Analysis

* Top Customers
* Deal Size Distribution

---

## Project Workflow

1. Import raw sales dataset
2. Clean and preprocess data using Python
3. Export cleaned dataset
4. Import cleaned dataset into Power BI
5. Create KPIs and measures
6. Build interactive dashboards
7. Generate automated business reports

---

## Expected Outcomes

* Improved data quality and consistency
* Reduced manual data cleaning effort
* Automated reporting process
* Enhanced decision-making through visual analytics
* Better understanding of data preprocessing and reporting automation
